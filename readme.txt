== Description ==

wp e-commerce best sellers plugin implement a widget that you can show Best Sellers Products of your shopping store.

== Installation ==

1. Upload the folder 'wp-e-commerce-best-sellers' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Setting  ==
you may need to add a css class wpsc_best_sellers to set best sellers list style.
